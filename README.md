# Nexty

Nexty is a task and project management application inspired by Jira and ClickUp.  
It helps teams plan, track, and organize their work efficiently.

## Overview
Nexty allows you to:
- Create and manage tasks and projects
- Organize work using boards (Kanban) or lists
- Track progress and deadlines
- Collaborate with team members

This is a demo/first version, aiming to experiment with task management features and project workflows.

## Features
- Task creation with unique IDs
- Task assignment and status tracking
- Boards and list views
- Simple reporting and activity overview
- Git workflow experiments integrated for development tasks
